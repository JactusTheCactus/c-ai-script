# Instructions
1. Run `script.py` to build readable versions of Chat Logs & Lore Books.
1. file names follow a specific format:
    - `Chats`: 
        - `User&MainCharacter-##*optional.json/jsonl`
        - example: `Johnny&Jessica-03.json`
    - `Lore Books`:
        - `topic.json`
        - example: `Lakedaimon.json`
